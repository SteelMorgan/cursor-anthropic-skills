## [1.2.0] - 2025-01-XX

### Добавлено
- **Development Methodology Rule** - Методика разработки SDD/TDD/DDD для проектов 1С/BSL
- **YAxUnit Testing Skill** - Навык для тестирования 1C приложений с использованием YAxUnit
- **Go Language Skill** - Навык для разработки на языке Go
- **Mermaid Skill** - Навык для создания диаграмм Mermaid
- Интеграция с Claude Code Templates репозиторием
- Обширная коллекция Claude Agent Skills (100+ навыков)
- Обновленная документация по установке глобального правила в Cursor
