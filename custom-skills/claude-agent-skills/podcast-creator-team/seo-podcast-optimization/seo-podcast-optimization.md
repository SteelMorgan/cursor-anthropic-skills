# SEO Podcast Optimization

This skill provides comprehensive guidance for creating SEO-friendly titles, meta descriptions, and identifying relevant keywords for podcast episodes.

## Overview

SEO podcast optimization involves crafting search-optimized content that balances keyword effectiveness with engaging copy. This skill provides systematic approaches to SEO optimization as an SEO Podcast Optimization Specialist.

## ✓ Pre-Optimization Checklist

**BEFORE optimizing podcast SEO, verify:**

```yaml
Checklist:
  - [ ] Episode content or summary available
  - [ ] Target keywords identified
  - [ ] Title requirements understood
  - [ ] Meta description length limits known
  - [ ] Keyword research completed
  - [ ] Optimization objectives defined

Status: [READY / BLOCKED]
```

**IF ANY CHECKBOX UNCHECKED:**
- Action: STOP
- Reason: Missing information required for effective SEO optimization

**IF ALL CHECKED:**

## Core Tasks

1. **Analyze Content**: Extract key themes, technologies, and concepts
2. **Create SEO-Optimized Title**: Craft compelling title <= 60 characters with primary keywords
3. **Write Meta Description**: Create concise description <= 160 characters with value proposition
4. **Identify Long-Tail Keywords**: Propose 3 long-tail keywords (3-5 words each) with search volume and relevance

## Output Format

Provide structured SEO optimization report with:
- Optimized Title (with character count)
- Meta Description (with character count)
- Long-Tail Keywords (with estimated volume and relevance scores)
- Rationale for keyword selection strategy

## Best Practices

- Balance keyword effectiveness with engaging copy
- Ensure titles are click-worthy while maintaining accuracy
- Include clear value proposition in descriptions
- Focus on specific tech concepts and solutions
- Provide rationale for keyword choices

## Tools and Methods

- **Read**: Analyze episode content and summaries
- **Write**: Create SEO-optimized titles and descriptions
- **WebSearch**: Research keyword search volumes and trends
