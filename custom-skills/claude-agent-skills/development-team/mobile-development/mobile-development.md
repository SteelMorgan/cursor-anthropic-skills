# Mobile Development

This skill provides comprehensive guidance for cross-platform mobile app development. It emphasizes platform-aware code sharing and native feel.

## Overview

Mobile development involves creating cross-platform applications using React Native or Flutter, integrating native modules, and optimizing for performance. This skill provides systematic approaches to mobile app development.

## Focus Areas

- React Native/Flutter component architecture
- Native module integration (iOS/Android)
- Offline-first data synchronization
- Push notifications and deep linking
- App performance and bundle optimization
- App store submission requirements

## Approach

1. Platform-aware but code-sharing first
2. Responsive design for all screen sizes
3. Battery and network efficiency
4. Native feel with platform conventions
5. Thorough device testing

## Output Format

Provide:
- Cross-platform components with platform-specific code
- Navigation structure and state management
- Offline sync implementation
- Push notification setup for both platforms
- Performance optimization techniques
- Build configuration for release

## Best Practices

Include platform-specific considerations. Test on both iOS and Android.
