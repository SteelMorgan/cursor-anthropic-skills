# Podcast Metadata

This skill provides comprehensive guidance for creating SEO-optimized podcast metadata, show notes, and platform-specific descriptions.

## Overview

Podcast metadata creation involves generating titles, descriptions, chapter markers, and platform-specific content. This skill provides systematic approaches to podcast metadata as a Podcast Metadata Specialist.

## Core Tasks

- Generate compelling, SEO-optimized episode titles
- Create detailed timestamps with descriptive chapter markers
- Write comprehensive show notes
- Extract memorable quotes and key takeaways
- Generate relevant tags and categories
- Create platform-optimized social media post templates
- Format descriptions for various podcast platforms

## Platform-Specific Requirements

- **YouTube**: Maximum 5000 characters, clickable timestamps (MM:SS or HH:MM:SS), optimize for YouTube search
- **Apple Podcasts**: Maximum 4000 characters, clean text formatting, focus on episode value proposition
- **Spotify**: HTML formatting supported, emphasis on listenability and engagement

## Quality Standards

- Titles should be 60-70 characters for optimal display
- Descriptions must hook listeners within first 125 characters
- Chapter titles should be action-oriented and descriptive
- Tags should include both broad and niche terms
- Social media posts must be engaging with relevant hashtags
- All timestamps must be accurate and properly formatted

## Best Practices

- Optimize for both human readers and search algorithms
- Ensure consistency across all metadata elements
- Focus on engagement and discoverability
- Maintain highest standards of quality
- Prioritize accuracy and optimization

## Tools and Methods

- **Read**: Analyze podcast content, transcripts, and requirements
- **Write**: Create metadata files, show notes, and descriptions
