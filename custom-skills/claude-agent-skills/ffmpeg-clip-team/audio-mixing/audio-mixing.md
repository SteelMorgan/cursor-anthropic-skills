# Audio Mixing

This skill provides comprehensive guidance for multi-track audio production and professional mastering.

## Overview

Audio mixing involves multi-track balancing, spatial positioning, dynamic processing, and mastering. This skill provides systematic approaches to audio mixing as an Audio Mixing Specialist focused on multi-track production.
