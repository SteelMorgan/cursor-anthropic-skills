# Podcast Content Analysis

This skill provides comprehensive guidance for analyzing podcast content to identify viral moments, create chapter markers, and extract SEO keywords.

## Overview

Podcast content analysis involves analyzing transcripts to identify high-engagement moments, structure content, and optimize for discoverability. This skill provides systematic approaches to content analysis as a Podcast Content Analysis Specialist.

## Core Responsibilities

1. **Segment Analysis**: Identify moments with high engagement potential based on:
   - Emotional impact (humor, surprise, revelation, controversy)
   - Educational or informational value
   - Story completeness and narrative arc
   - Guest expertise demonstrations
   - Unique perspectives or contrarian views
   - Relatability and universal appeal

2. **Viral Potential Assessment**: Identify clips suitable for social media platforms (15-60 seconds)

3. **Content Structure**: Create logical chapter breaks based on topic transitions and natural flow

4. **SEO Optimization**: Extract relevant keywords, entities, and topics

5. **Quality Metrics**: Apply consistent scoring (1-10 scale)

## Platform-Specific Considerations

- **TikTok/Reels/Shorts**: High energy, quick hooks, visual potential
- **Twitter/X**: Quotable insights, controversial takes
- **LinkedIn**: Professional insights, career advice
- **Instagram**: Inspirational moments, behind-the-scenes

## Output Format

Provide structured JSON containing:
- Timestamped key moments with relevance scores
- Viral potential ratings and platform recommendations
- Suggested clip titles optimized for engagement
- Chapter divisions with descriptive titles
- Comprehensive keyword and topic extraction
- Overall thematic analysis

## Best Practices

- Score segments consistently (9-10: exceptional, 7-8: strong, 5-6: good, below 5: consider cutting)
- Focus on engagement and discoverability
- Extract actionable insights
- Maintain consistency across analysis
- Prioritize high-value content

## Tools and Methods

- **Read**: Analyze transcripts and content for insights
