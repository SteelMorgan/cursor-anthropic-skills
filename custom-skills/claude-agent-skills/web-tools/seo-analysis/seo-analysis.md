# SEO Analysis

This skill provides comprehensive guidance for technical SEO audits, content optimization, and search engine performance improvements.

## Overview

SEO analysis involves conducting technical SEO audits, optimizing meta tags, analyzing performance, and providing search engine optimization recommendations. This skill provides systematic approaches to SEO analysis as an SEO analysis specialist focused on technical SEO audits.

## ✓ Pre-Analysis Checklist

**BEFORE starting SEO analysis, verify:**

```yaml
Checklist:
  - [ ] Website URL and structure identified
  - [ ] Target keywords and goals defined
  - [ ] Current SEO metrics collected
  - [ ] Competitor analysis planned
  - [ ] Technical audit scope determined
  - [ ] Performance baseline established

Status: [READY / BLOCKED]
```

**IF ANY CHECKBOX UNCHECKED:**
- Action: STOP
- Reason: Missing information required for effective SEO analysis
- Next step: Gather missing information before proceeding

**IF ALL CHECKED:**
- Output: "✅ Pre-analysis verification complete: Ready for SEO analysis"
- Proceed with: SEO analysis workflow

## Focus Areas

- **Technical SEO audits** and site structure analysis
- **Meta tags**, titles, and description optimization
- **Core Web Vitals** and page performance analysis
- **Schema markup** and structured data implementation
- **Internal linking** structure and URL optimization
- **Mobile-first indexing** and responsive design validation

## Approach

1. Comprehensive technical SEO assessment
2. Content quality and keyword optimization analysis
3. Performance metrics and Core Web Vitals evaluation
4. Mobile usability and responsive design testing
5. Structured data validation and enhancement
6. Competitive analysis and benchmarking

## Output Deliverables

- Detailed SEO audit reports with priority rankings
- Meta tag optimization recommendations
- Core Web Vitals improvement strategies
- Schema markup implementations
- Internal linking structure improvements
- Performance optimization roadmaps

## Best Practices

- Focus on actionable recommendations
- Improve search rankings and user experience
- Include specific implementation examples
- Provide expected impact metrics
- Prioritize high-impact improvements

## Tools and Methods

- **Read**: Analyze website structure, HTML, and configurations
- **Write/Edit**: Optimize meta tags, implement schema markup, improve content
- **WebFetch**: Fetch and analyze web pages
- **Grep**: Search for SEO-related patterns
- **Glob**: Find files for SEO analysis
