# Customer Support

This skill provides comprehensive guidance for quick resolution and customer satisfaction. It emphasizes empathy, clear solutions, and follow-up.

## Overview

Customer support involves responding to tickets, creating documentation, troubleshooting issues, and ensuring customer satisfaction. This skill provides systematic approaches to customer support.

## Focus Areas

### Support Ticket Responses

- Empathetic acknowledgment
- Clear step-by-step solutions
- Screenshots when helpful
- Alternative solutions if blocked
- Follow-up on resolution

### FAQ Documentation

- Common questions
- Clear answers
- Searchable format
- Regular updates

### Troubleshooting Guides

- Step-by-step procedures
- Visual aids
- Common issues
- Resolution steps

### Canned Response Templates

- Standard responses
- Personalization guidelines
- Escalation criteria
- Follow-up templates

### Help Center Articles

- Comprehensive documentation
- Search functionality
- Visual guides
- Video tutorials

### Customer Feedback Analysis

- Feedback collection
- Analysis and insights
- Action items
- Improvement tracking

## Approach

1. Acknowledge the issue with empathy
2. Provide clear step-by-step solutions
3. Use screenshots when helpful
4. Offer alternatives if blocked
5. Follow up on resolution

## Output Format

Provide:
- Direct response to customer issue
- FAQ entry for common problems
- Troubleshooting steps with visuals
- Canned response templates
- Escalation criteria
- Customer satisfaction follow-up

## Best Practices

- Keep tone friendly and professional
- Always test solutions before sharing
- Document common issues
- Track resolution times
- Monitor customer satisfaction
